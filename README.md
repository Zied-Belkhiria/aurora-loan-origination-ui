# AuroraLoan UI

Angular frontend for the AuroraLoan Loan Origination backend.

## Features
- Customers: create + fetch by ID
- Accounts: create + fetch by ID
- Loans: create & evaluate (PL/SQL decision) + fetch by ID

## Setup
```bash
npm install
npm start
```

Backend should run on:
`http://localhost:8080`
